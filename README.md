# Chef's Secret 

A responsive recipe website built with HTML, CSS, JavaScript, and Bootstrap that showcases delicious recipes with modern web features.

##  Features

### Core Functionality
- **Recipe Catalog** - Browse recipes by categories (Breakfast, Lunch, Drinks, Salad, Dessert)
- **Search System** - Find recipes with autocomplete suggestions
- **Responsive Design** - Mobile-first design that works on all devices
- **Dark/Light Mode** - Toggle between themes with persistent settings
- **Sign in/ Sign up** - You can sign in and sign up and look at your profile

### Interactive Elements
- **Text Highlighting** - Highlight search terms across FAQ pages
- **Star Rating System** - Interactive rating with scroll animations
- **Contact Form** - Form validation with loading states
- **FAQ Accordion** - Collapsible question-answer sections
- **Scroll Progress Bar** - Visual scroll indicator
- **Popup Subscription** - Newsletter signup with sound effects

### Technical Features
- **Lazy Loading Images** - Optimized image loading for performance
- **Keyboard Navigation** - Navigate menu with arrow keys
- **Local Storage** - Remember user preferences (theme)
- **Animated Counters** - Statistics with counting animations
- **Copy to Clipboard** - Copy recipe ingredients with one click

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Frameworks**: Bootstrap 5.3, jQuery
- **Icons**: Bootstrap Icons
- **Tools**: Git, GitHub
